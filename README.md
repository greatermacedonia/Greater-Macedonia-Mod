# Greater Macedonia

Greater Macedonia is an alternate-history mod for Hearts of Iron IV where Macedonia rises to reclaim its ancient legacy and shape the fate of the Balkans and beyond.

Steam Workshop:
https://steamcommunity.com/sharedfiles/filedetails/?id=3654937421

## Overview

This mod adds custom Macedonian gameplay with a dedicated focus tree, event chains, and decision content covering both historical inspiration and ahistorical paths.

## Features

- New Macedonian national focus tree with multiple political and strategic routes.
- Custom events and decisions centered on ancient legacy, revolutionary movements, and territorial ambitions.
- Unique Macedonian leaders, including Goce Delcev, Metodija Andonov-Cento, and Mihailo Apostolski.
- Original ideas and opinion modifiers for diplomacy and internal development.
- Custom graphics including leader portraits and military portrait assets.

## Compatibility And Requirements

- The mod is currently work in progress.
- No specific DLC is required.
- Some features may not fully integrate with all DLC content.
- Best experience is with current Hearts of Iron IV versions; balance and compatibility updates are ongoing.

## Known Issues

- Some focus tree branches and events are still under active development.
- Minor localisation gaps can still appear.
- A few ideas or decisions may show code names while final polish is in progress.

## Current Direction

The project continues to expand in these areas:

- Additional focus tree branches and alternate-history routes.
- Deeper diplomacy and territorial-demand event chains.
- Ongoing balance and compatibility updates.
- Additional localisation support.
- More custom art and interface assets.

## Repository Structure

- common/: countries, ideas, focus trees, decisions, leaders, units, and script data.
- events/: event chains for legacy, diplomacy, crises, and historical variants.
- history/: state, country, and unit setup.
- localisation/english/: English localisation entries.
- gfx/: portraits, flags, ideas, and interface artwork.
- interface/: portrait and idea sprite definitions.
- map/: map support data.
- descriptor.mod: mod descriptor file.
